# planets

## Dependencies
### Arch
```
pacman -S sfml cmake
```
###Debian
```
sudo apt-get install libsfml-dev
```
### More info
```
https://www.sfml-dev.org/tutorials/2.6/start-linux.php
```

## Quick Start
```
# cd into src

# First Time Cmake Setup
cmake -S . -B ./build

# Build
cmake --build ./build

# Run
./build/sfml-app
```
